Variáveis Nativas CSS Versus Variáveis SASS

As variáveis ​​são uma das principais razões pelas quais os pré-processadores de CSS existem. A capacidade de definir uma variável para algo como uma cor, usar essa variável em todo o CSS que você escreve e saber que ela será consistente, seca e fácil de alterar é útil. Você pode usar variáveis ​​CSS nativas (CSS Custom Properties) pelas mesmas razões. Mas também existem algumas diferenças importantes que devem ficar claras.

Vantagens do uso das variáveis nativas do CSS

  • Evita a repetição de código e melhora a manutenção deste código;
  • Você pode usá-las sem a necessidade de um pré-processador;
  • Suporte à manipulação dinâmica: podem ser mudadas em tempo de execução;
  • Conceito de escopo: você pode definir uma variável dentro de qualquer seletor para definir ou substituir seu valor atual;
  • Você pode acessá-las e manipulá-las com JavaScript.
